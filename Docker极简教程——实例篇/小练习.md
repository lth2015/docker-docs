### 试验环境
------------------------------------------------------------------------------------------
##### 打开控制台，登录试验环境
```bash
ssh docker@172.21.1.11
```

##### 切换至实验目录docker, 创建自己名字的的目录
```bash
cd ~/docker && mkdir dawei
```
